# 《深度学习.md》排版布局与自学敲写说明

## 一、排版布局规划

这份笔记建议采用“从基础概念到网络训练，再到 CNN 推理结构”的顺序，而不是把概念平铺在一起。整体结构如下：

```text
# 深度学习
  ## 目录
  ## 神经网络
    ### 感知机
    ### 激活函数
    ### 多维数组的运算
    ### 3层神经网络的实现
    ### 输出层设计
  ## 神经网络的学习
    ### 损失函数
    ### 梯度
    ### 学习算法的实现
  ## 误差反向传播法
    ### 计算图
    ### 链式法则
    ### 反向传播
    ### 简单层
    ### 激活层
    ### Affine/Softmax层
    ### 误差反向传播的实现
  ## 与学习相关的技巧
    ### 参数的更新
    ### 权重的初始值
    ### Batch
    ### 正则化
    ### 超参数的验证
  ## 卷积神经网络
    ### 卷积层
    ### 池化层
    ### BN、激活函数与推理
    ### 卷积与 BN 融合
    ### CNN 学习时要抓住的主线
```

这样的布局有三点好处：

1. 先理解神经网络“怎么算”，再理解“怎么学”。
2. 先理解损失函数和梯度，再学习反向传播，不会把反向传播背成孤立公式。
3. 最后进入 CNN，把卷积、BN、激活函数和推理加速联系起来，方便后续接 FPGA/HLS 加速内容。

## 二、标题层级规则

建议只使用三级标题。

- `#`：整份笔记标题，只出现一次。
- `##`：大知识模块，例如神经网络、反向传播、CNN。
- `###`：模块内的具体知识点，例如损失函数、梯度、卷积层。

尽量不要频繁使用四级标题。需要细分时，可以用列表、公式和短段落来组织。

## 三、每个知识点的固定写法

每个小节建议按以下顺序写：

1. 先写一句“这个概念是什么”。
2. 再写核心公式。
3. 接着解释公式里每个符号的含义。
4. 最后写一句“它在神经网络中有什么作用”。

例如“损失函数”可以这样写：

```md
### 损失函数

损失函数用于度量预测结果和真实标签之间的差距。

$$
L = -\sum_i t_i \log y_i
$$

其中，$y_i$ 是预测概率，$t_i$ 是真实标签的 one-hot 表示。
分类任务中，交叉熵损失越小，说明模型给真实类别的概率越高。
```

这种写法比只抄公式更适合复习，因为以后看笔记时能快速知道公式的用途。

## 四、需要自己敲的内容

下面这些内容建议你自己在笔记里亲手敲一遍，不要只复制。

### 1. 维度标注

在全连接层公式旁边敲出维度：

```md
若 $X \in \mathbb{R}^{N \times D}$，$W \in \mathbb{R}^{D \times H}$，
则 $Y = XW + b$ 的输出维度为 $Y \in \mathbb{R}^{N \times H}$。
```

自己敲维度能训练你检查矩阵乘法是否合理。

### 2. 激活函数图像或输入输出例子

建议自己补一个小表：

```md
| 输入 x | ReLU(x) | LeakyReLU(x) |
| ------ | ------- | ------------ |
| -2     | 0       | -0.02        |
| 0      | 0       | 0            |
| 3      | 3       | 3            |
```

这样能直观看出 ReLU 和 Leaky ReLU 的区别。

### 3. 反向传播的链式法则推导

建议自己敲一遍：

```md
若 $y=f(x)$，$L=g(y)$，则：

$$
\frac{\partial L}{\partial x}
=
\frac{\partial L}{\partial y}
\frac{\partial y}{\partial x}
$$
```

这一条是理解反向传播的核心。

### 4. Softmax + 交叉熵的结果

建议记住并手敲：

```md
Softmax 和交叉熵损失组合后，反向传播结果为：

$$
\frac{\partial L}{\partial z_i} = y_i - t_i
$$
```

这条公式在理解分类网络训练时非常重要。

### 5. 卷积输出尺寸公式

建议自己敲并代入一个例子：

```md
$$
H_{\text{out}}
=
\left\lfloor
\frac{H_{\text{in}} + 2P - R}{S_{\text{stride}}}
\right\rfloor + 1
$$
```

可以代入：输入尺寸 $32 \times 32$，卷积核 $3 \times 3$，padding 为 1，stride 为 1，输出仍然是 $32 \times 32$。

### 6. BN 融合公式

这部分和后续 FPGA 推理加速关系很大，建议自己敲一遍：

```md
$$
y = az + c
$$

$$
a = \frac{\gamma}{\sqrt{\sigma^2+\epsilon}}
$$

$$
c = \beta - \frac{\gamma \mu}{\sqrt{\sigma^2+\epsilon}}
$$

$$
W' = aW
$$

$$
b' = ab + c
$$
```

需要特别记住：BN 融合主要用于推理阶段，不是训练阶段。

## 五、建议补充的小练习

可以在笔记最后加一个“自测”区域，每学完一节就回答 2 到 3 个问题。

```md
## 自测

1. 为什么没有激活函数的多层神经网络仍然等价于线性模型？
2. 交叉熵损失为什么适合分类任务？
3. 反向传播中“上游梯度”和“局部梯度”分别是什么？
4. 卷积层相比全连接层为什么参数更少？
5. BN 为什么可以在推理阶段和卷积融合？
```

如果能用自己的话回答这些问题，说明这份笔记已经从“整理内容”变成了“真的学懂”。

## 六、后续扩展方向

这份笔记目前适合作为深度学习基础笔记。后续如果要服务 YOLO、FPN 和 FPGA/HLS，可以继续扩展三类内容：

1. CNN 到 YOLO：补充卷积特征图、检测头、边界框、类别预测、NMS。
2. YOLO 到 FPN：补充多尺度特征、上采样、横向连接、特征融合。
3. 模型到硬件：补充定点量化、卷积循环展开、line buffer、window buffer、PE 阵列和数据复用。

扩展时仍然建议保持“概念 -> 公式 -> 作用 -> 和硬件/模型的关系”的格式。
