# A2 Feature Entropy Measurement Result

Experiment ID:
A2_entropy_pruning

Experiment stage:
Stage 1 - YOLO Baseline Establishment and Entropy-Guided FPN Pruning

Experiment objective:
Measure the feature-map spatial information entropy of YOLOv8n Neck/FPN/PAN fusion-refinement layers and provide evidence for later pruning decisions.

Baseline checkpoint:
07_experiments/stage1_algorithm_baseline_pruning/A1_yolov8n_baseline_100ep/weights/best.pt

Dataset:
03_dataset/yolo_format/neu_det.yaml

Split:
val

Image count:
64

Measured layers:
12, 15, 18, 21

Entropy method:
For each selected layer, the output tensor is treated as BxCxHxW. Each channel map is normalized to [0, 1], converted to a 256-bin histogram, and measured by spatial entropy:

H_c = -sum_b p_c(b) * log2(p_c(b) + eps)

Layer-level entropy:
H_bar = mean_c(H_c)

Epsilon:
1e-8

Conclusion:
The entropy-only lowest layer is layer 21. However, because layer 21 also has the smallest activation footprint, it is not the strongest hardware-aware pruning target. Considering both feature entropy and activation movement cost, layer 18 is selected as the primary candidate for the next pruning-design step. Layer 21 and layer 15 should be retained as control candidates for ablation-style comparison.
