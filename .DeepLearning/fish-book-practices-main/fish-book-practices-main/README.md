# 深度学习入门(鱼书) 配套练习

本项目是《深度学习入门：基于Python的理论与实现》（俗称"鱼书"）的配套代码实现，包含了从基础感知机到卷积神经网络的完整深度学习实现。

## 📚 项目简介

这是一个从零开始实现深度学习算法的教学项目，不依赖深度学习框架，使用纯NumPy实现神经网络的各个组件。通过本项目，你可以深入理解深度学习的底层原理。

## 🚀 快速开始

### 环境要求

- Python >= 3.12


### 安装依赖

使用uv包管理器（推荐）：
```bash
uv sync
```
